Sistemas Operacionais

Oi gente!

Hoje eu quero trocar uma ideia com vocês sobre sistemas operacionais.

Quando você liga seu computador, alguma coisa precisa controlar como seu teclado, sua placa de vídeo, seu mouse, memória, processador, etc., devem se comportar. O nome do programa que faz isso é sistema operacional.

Um dos primeiros sistemas operacionais – e o mais importante da história – foi o Unix, criado na década de 1970 no Bell Labs, por Dennis Ritchie e Ken Thompson.

A partir do Unix, surgiram o IEEE Std. 1003.1™ (Posix), que define a interface programática para interagir com sistemas operacionais, e o ELF, formato padrão de executáveis.

Quase todos os sistemas operacionais modernos têm alguma relação com o Unix e/ou Posix.

Microsoft Windows™  

Windows é muito provavelmente o sistema operacional que você está usando agora para acessar esta página. Foi criado na década de 1980 pela Microsoft uma tentativa de concorrer com os sistemas operacionais da Apple.

O Windows nasceu como uma interface gráfica sobre o MS-DOS. Lá pela década de 1990, foi lançada uma versão autônoma do Windows que todos conhecem por “série 9x” (Windows 95, Windows 98, Windows 98SE e Windows ME).

Mas o Windows não parou por aí: a Microsoft participava de um projeto com a IBM para o desenvolvimento de um sistema operacional chamado OS/2 Warp. A Microsoft se apropriou dos códigos e desenvolveu um novo sistema por conta própria, chamado Windows NT, que implementa parcialmente o padrão Posix.

Todas versões posteriores do Windows – NT4, Windows 2000, Windows 2003, Windows XP, Windows Vista e Windows 7 em diantes –, com exceção do Windows ME, são versões do NT.

BSD  󰣠

Outro sistema operacional muito importante é o BSD, Berkeley Software Distribution, criado na Universidade de Berkeley a partir dos próprios códigos do Unix.

Há diversas distribuições do USB, que são “empacotamentos” que facilitam a instalação e o uso do sistema por usuários doméstricos, no geral gratuitas.

Algumas distribuições importantes são:

  • FreeBSD  
  • OpenBSD  
  • 386BSD
  • DragonFly BSD
  • NetBSD

Mach Kernel

A partir do BSD, foi criado o Mach Kernel, o kernel (cerne do sistema) do NeXTStep, da empresa NeXT de Steve Jobs, que seria adquirida pela Apple na década de 1990.

macOS  

Os primeiros sistemas operacionais da Apple foram o Lisa e o Mac OS.

Quando a Apple adquiriu a NeXT e o NeXTStep, a empresa adaptou este para rodar em seus computadores MacIntosh, incorporando elementos do Mac OS. O novo sistema foi chamado OS X. Mais tarde o sistema viria a ser renomeado “macOS”.

GNU/Linux  

Na década de 1990, um programador finlandês chamado Linus Torvalds estava trabalhando numa versão do Minix que pudesse ser usada em produção, uma vez que o Minix é apenas um kernel de estudo e experimentação. O objetivo era usá-lo no sistema operacional GNU, da Free Software Foundation.

O kenel recebeu o nome “Linux”, e o sistema operacional GNU com esse kernel é chamado “GNU/Linux”.

Hoje em dia você praticamente não encontra um servidor que não rode GNU/Linux, nem um programador sério que não trabalhe com ele.

Assim como o BSD, o GNU/Linux é distribuído em “distribuições”, empacotamentos do sistema que facilitam sua instalação.

Seria impossível citar todas as distribuições num artigo pequeno, mas algumas das mais importantes são (em ordem alfabética):

  • Arch Linux  
  • CentOS  
  • Debian GNU/Linux  
  • Fedora Core  
  • Manjaro  
  • Open SuSE Linux  
  • RedHat Enterprise Linux  
  • Slackware  
  • Ubuntu  
