Instalando Python

Para nossos exemplos e exercícios usaremos Python como linguagem de programação na maioria das vezes, portanto é importante que tenhamos ela instalada.

Distruições GNU/Linux baseadas em apt

Vamos começar instalando Python no Debian GNU/Linux, que usado como gerenciador de pacotes o apt. Outras distros, como o Ubuntu e seus variantes, também utilizam o apt.

Para buscar por um pacote usamos o subcomando search. Para Buscar por Python 3.11, podemos executar o seguinte comando:

apt search python3.11

Isso retornará uma lista onde poderemos encontrar o Python, provavelmente algo mais ou menos assim:

python3.11-full/testing 3.11.4-1 amd64
  Python Interpreter with complete class library (version 3.11)

Na primeira linha vemos o nome do pacote (python3.11-full), a versão do sistema (testing), a versão do pacote (3.11.4-1) e a arquitetura (amd64).

Para instalar esse pacote usamos o subcomando install, no entanto precisamos prefixar todo o comando com sudo, que o executará como usuário administrador (root):

sudo apt install -y python3.11-full

Provavelmente pedirá a senha do seu usuário e, se seu usuário for sudoer (que pode requerer permissões de adminstrador), então o CPython (interpretador Python escrito em C) será instalado. Você pode confirmar isso após o final da instalação executando o comando python.

Para sair do interpretador Python basta pressionar Control+D (C-d no jargão) ou executar exit().

Para nossos testes, usaremos um REPL1 alternativo com uma série de funções extras, como auto-complete, chamado Ptpython. É fácil instalá-lo no Debian:

sudo apt install -y ptpython

Para gerência de versionamento dos códigos, vamos usar o Git, também facilmente instalável:

sudo apt install -y git

Você também pode instalar todos esses pacotes em um só comando:

sudo apt install -y python3.11-full ptpython git

Distruições GNU/Linux baseadas em pacman

Outra família de distribuições GNU/Linux importante é a do Arch Linux, que utiliza o gerenciador de pacotes pacman.

No entanto não é tão simples quanto usando apt, apesar de também não ser ciência de foguete.

Para instalar o Python, vamos ter de instalar três pacotes, mais o Git:

sudo pacman -Syu python python-pip python-pipx git

Os parâmetros -Syu são os parâmetros de instalação, o pacote python é o Python 3.11 (talvez já tenha sido atualizado), python-pip é o gerenciador de pacotes de Python e python-pipx é o gerenciador de pacotes extras.

Agora vamos instalar o Ptpython, que pode ser apenas para o usuário, não de sistema. Para tanto usaremos o pipx:

pipx install ptpython

Windows

Para instalar Python, Ptpython e Git no Windows vamos usar um gerenciador de pacotes chamado Scoop.

Para instalar o Scoop, abra o PowerShell e executa os comandos:

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex

Instalado o Scoop, buscamos por Python com:

scoop search python

Vai aparecer Python 3.11 na lista, que provavelmente será o pacote python, então, para instalar:

scoop install python

No Windows não precisamos do pipx para instalar o Ptpython, podemos instalá-lo com o próprio pip:

pip install ptpython

E por último o Git:

scoop install git

Bônus: Distruições GNU/Linux baseadas em RPM

Outra família importante de distros GNU/Linux é RPM, onde econtramos o RHEL, CentOS e SuSE Linux.

Para instalar Python e Git nesses sistemas usamos o yum: 

yum install python3 git

No momento não há instruções de instalação do Ptpython em distros RPM, conforme explicado no PR #386 do Ptpython.
